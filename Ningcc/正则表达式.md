# 正则表达式

>

### 一、

* `.`可以匹配任何字符，

* 字符集匹配，

  ```javascript
  abc aoc adc afc akc 
  // 任意字符匹配
  a[bodfk]c ,可匹配以上
  ```

* 字母范围[a-z]

* 为了表示一个字符是可选的，我们在它后面加一个 ?。例如，表示下方文本中的字母 u 是可选的

  ```shell
  color, colour
  #colou?r
  ```

* 用 `*` 匹配下方文本中，没有或存在多个字母 e 的单词。

  dp dep deep

  ```shell
  de*p
  ```

* 用 `+` 匹配下方文本中，字母 e 出现一次或多次的单词。

  dp dep deep

  ```shell
  de+p   #dep deep
  ```

* 用 ? 表示字母 n 在文本中是可选的，使 a 和 an 都可以匹配到。

  a, an

  ```shell
  an?  #a, an
  ```

* \W 匹配除字母、数字和下划线之外的字符

* \w 匹配字母、数字和下划线之外的字符
* \s 仅匹配空白字符

* 我们要匹配文本中的小时值。为了只匹配后面有 PM 的数值，我们需要在表达式后面使用正向先行断言 (?=)，并在括号内的 = 后面添加 PM。

  ```shell
   Date: 4 Aug 3PM
  ```

  ```shell
  \d+(?=PM)    # 匹配第二个数字3
  ```
